Enhancer\n------------------\n\n[](https://github.com/salvogiangri/KnoxPatch#knoxpatch-enhancer)\n\nKnoxPatch Enhancer is a flashable zip that will take care of the (currently) non-fixable apps/features via the Xposed API's. There are two different install modes:\n\n*   Via the [Magisk](https://github.com/topjohnwu/Magisk)\n    /[KernelSU](https://github.com/tiann/KernelSU)\n     app (fix Galaxy Wearable apps/Smart View):\n    \n    Download the module zip from the [latest release](https://github.com/salvogiangri/KnoxPatch/releases/latest)\n     and install it from the \"Modules\" tab inside the Magisk/KernelSU app. The app will then show an \"Enhanced\" badge if the module is installed and active:\n    \n\n[![](https://github.com/salvogiangri/KnoxPatch/raw/main/readme-res/kp-enhancer.png)](https://github.com/salvogiangri/KnoxPatch/blob/main/readme-res/kp-enhancer.png)\n\n*   Via a custom recovery (fix Secure Folder on legacy devices):\n    \n    Legacy devices running One UI 2.x or 3.x with [FBE](https://source.android.com/docs/security/features/encryption/file-based)\n     require additional patches in the system partition to fix Secure Folder. Download the module zip from the [latest release](https://github.com/salvogiangri/KnoxPatch/releases/latest)\n     and install it via a custom recovery, this will apply the necessary patches to fix Secure Folder. The zip will also create a backup of your original system files, flashing it again will restore them.\n    \n\nCredits\n-------\n\n[](https://github.com/salvogiangri/KnoxPatch#credits)\n\n*   Samsung\n*   [LSPosed Team](https://github.com/LSPosed)\n    \n*   [YukiHookAPI](https://github.com/fankes/YukiHookAPI)\n    \n*   [Rikka](https://github.com/RikkaApps)\n    \n\nCopyright\n---------\n\n[](https://github.com/salvogiangri/KnoxPatch#copyright)\n\n    /*\n     * Copyright (C) 2022 Salvo Giangreco\n     *\n     * This program is free software: you can redistribute it and/or modify\n     * it under the terms of the GNU General Public License as published by\n     * the Free Software Foundation, either version 3 of the License, or\n     * (at your option) any later version.\n     *\n     * This program is distributed in the hope that it will be useful,\n     * but WITHOUT ANY WARRANTY; without even the implied warranty of\n     * MERCHANTABILITY or FITNESS FOR A PARTICULAR PURPOSE.
