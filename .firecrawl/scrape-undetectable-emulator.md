items\n\nUndetected Android Emulator\n===========================\n\n[](https://github.com/agentbryce2025/emulator#undetected-android-emulator)\n\nThis project aims to create an Android phone emulator that is undetectable by LinkedIn and other applications while providing a fully functional visual GUI experience.\n\nQuick Setup Guide\n-----------------\n\n[](https://github.com/agentbryce2025/emulator#quick-setup-guide)\n\n### Prerequisites\n\n[](https://github.com/agentbryce2025/emulator#prerequisites)\n\n*   Python 3.8 or higher\n*   QEMU 6.0+ (see platform-specific instructions below)\n\n### Simple Installation (All Platforms)\n\n[](https://github.com/agentbryce2025/emulator#simple-installation-all-platforms)\n\n1.  Run the Python setup script:\n    \n    ```shell\n    # On all platforms\n    python install_requirements.py\n    ```\n    \n2.  Activate the virtual environment:\n    \n    ```shell\n    # On Windows\n    venv\\Scripts\\activate\n    \n    # On macOS/Linux\n    source venv/bin/activate\n    ```\n    \n3.  Run the emulator:\n    \n    ```shell\n    python main.py\n    ```\n    \n\n### Alternative Platform-Specific Installation\n\n[](https://github.com/agentbryce2025/emulator#alternative-platform-specific-installation)\n\n#### Windows\n\n[](https://github.com/agentbryce2025/emulator#windows)\n\n1.  Download and install [QEMU for Windows](https://www.qemu.org/download/#windows)\n    \n2.  Run the setup script:\n    \n        setup.bat\n        \n    \n\n#### macOS\n\n[](https://github.com/agentbryce2025/emulator#macos)\n\n1.  Install dependencies:\n    \n    ```shell\n    # Using Homebrew\n    brew install python qemu libmagic\n    ```\n    \n2.  Run the setup script:\n    \n    ```shell\n    chmod +x setup.sh\n    ./setup.sh\n    ```\n    \n\n#### Linux (Ubuntu/Debian)\n\n[](https://github.com/agentbryce2025/emulator#linux-ubuntudebian)\n\n1.  Install dependencies:\n    \n    ```shell\n    sudo apt-get update\n    sudo apt-get install python3-pip python3-venv qemu-system-x86 qemu-utils libmagic1\n    ```\n    \n2.  Run the setup script:\n    \n    ```shell\n    chmod +x setup.sh\n    ./setup.sh\n    ```\n    \n\n### Command-line Options\n\n[](https://github.com/agentbryce2025/emulator#command-line-options)\n\n    python main.py --help\n    \n\n### Troubleshooting Common Issues\n\n[](https://github.com/agentbryce2025/emulator#troubleshooting-common-issues)\n\n#### Image Download Errors\n\n[](https://github.com/agentbryce2025/emulator#image-download-errors)\n\nIf you're experiencing timeouts or errors while downloading Android images:\n\n1.  **Use the 'Use Local File' Button**: The emulator now has a button to use a locally downloaded ISO file\n    \n2.  **Manual Download Instructions**:\n    \n    *   Go to the [Android-x86 Project Website](https://www.android-x86.org/download.html)\n        \n    *   Download the appropriate ISO file (e.g., android-x86\\_64-9.0-r2.iso)\n    *   Recommended: Android-x86 9.0-r2 for x86\\_64 architecture\n    *   Once downloaded, click \"Use Local File\" in the emulator's Images tab\n    *   Navigate to your download location and select the ISO file\n3.  **Direct Download Links**:\n    \n    *   [Android-x86 9.0-r2 (64-bit)](https://sourceforge.net/projects/android-x86/files/Release%209.0/android-x86_64-9.0-r2.iso/download)\n        \n    *   [Android-x86 8.1-r6
