(pdx203)\n\n[Raw](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4/raw/afe6909b84b47d94c644c602482fb176e4cd04f8/pdx203_a11.md)\n\n[**pdx203\\_a11.md**](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#file-pdx203_a11-md)\n\nThis is done on Android 11 for Xperia 1 II (pdx203) 58.1.A.5.395\n\nYou need to have bootloader unlocked and rooted via Magisk **(Not Magisk Delta, Alpha, Kitsune)**\n\nThis guide will assume you have already rooted. Root guide can be found somewhere on the internet.\n\nIn Magisk setting\n=================\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#in-magisk-setting)\n\n1.  Choose the option \"Hide the Magisk app\". Install it.\n2.  Enable Zygisk, reboot. Make sure that \"Enforce Denylist\" is disabled.\n3.  (Optional) Last line \"Superuser Notification\", set it to None to it won't send the toast notification on rooted apps.\n\nNecessary modules and apps\n==========================\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#necessary-modules-and-apps)\n\nInstall these modules with the following order:\n-----------------------------------------------\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#install-these-modules-with-the-following-order)\n\n1.  Zygisk - LSPosed\n2.  Shamiko\n3.  PlayIntegrityFix **(I haven't figured out with PlayIntegrityFork, let me know if you can)**\n4.  TrickyStore\n5.  TrickyStore Addons\n\nInstall these apps:\n-------------------\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#install-these-apps)\n\n1.  Applist Detector\n2.  KSUWebUI\n3.  Hide My Applist\n\nHide the root related apps from bank app (I use Vietinbank in this case)\n========================================================================\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#hide-the-root-related-apps-from-bank-app-i-use-vietinbank-in-this-case)\n\n1.  Install the Hide My Applist in LSPosed\n2.  Open the Hide My Applist. The version I used when this guide was being written was `V3.5r449...`\n3.  Open Template manage. You see there is an option \"Create a blacklist template\"\n4.  In Template settings, select \"Hide My Applist\", \"KsuWebUI\" and the fake Magisk app (normally Settings, has a random package name)\n5.  Go back to main menu. You see \"App manage\"\n6.  Select \"Applist Detector\" (and any bank app or apps that you need to hide root later on, need to be configured separately)\n7.  In the App settings, select \"Enable hide\". In the Template config, choose the template you do in step 4.\n8.  Reboot phone.\n9.  Open Applist Detector. All tests should be passed.\n10.  Open bank app and try to login. You should be good to go now.\n\nFix Integrity + Play Protect\n============================\n\n[](https://gist.github.com/B1ue-Dev/5f82ca09a94021f2086d3a4d4aa5d9f4#fix-integrity--play-protect)\n\n1.  Open KsuWebUI. You should see there is \"Play Integrity Fix\" and \"Tricky Store\"\n2.  Tap on Play Integrity Fix and choose \"Fetch pif.json\". Wait until it is done.\n3.  Go back and open Trick Store.\n4.  Open Tricky store : Open App menu \\[ Top right corner \\]\n\nSelect all > Deselect Unnecessary > Set valid Keybox > Save\n\nAgain open menu > Set security Patch > click \"Get security Patch date\" \\[ It'll auto configure the dates \\].\n\n**Remember to click Save**\n\n5.  Now, clear data/cache of these apps:\n\n*   Google Play Store\n*   Google Play services\n*   Google Services framework\n\n6.  Reboot the phone.\n7.  Check Integrity in Play Storeand Play Protect.
